# GlobalWomenRising

A platform for empowering women leaders worldwide.

## Quick Setup

1. Upload this folder to your VPS
2. Run the installation script:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
3. Start the server:
   ```bash
   ./start.sh
   ```

## Commands

- Start server: `./start.sh`
- Stop server: `./stop.sh`
- View at: `http://your-vps-ip:5000`

## Requirements

- Node.js 18+
- npm
